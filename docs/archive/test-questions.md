# ChatImage 测试问题集

本文件用于 ChatImage 的手工体验、自动化样例、真实 API smoke test 和产品验收测试。问题覆盖基础解释、流程、对比、时间线、矩阵、热点追问、技术架构、异常边界和真实内容等场景。

建议测试方式：

- 每次选择 1 个问题输入 ChatImage，检查是否生成结构化图片。
- 点击每个热点区域，检查详情是否匹配该区域。
- 在不同热点下分别追问，检查 thread 是否隔离。
- 刷新页面后检查最近记录是否保留。
- 优先挑选流程、对比、矩阵和真实内容类问题测试布局质量。

## 自动化结构化回归用例

固定测试集位于 `tests/structured-text-cases.json`，执行命令：

```powershell
npm.cmd run test:structured-text
```

当前固定 case 覆盖：

- `agent_workflow`：大模型 Agent 工作流程。
- `rest_graphql_compare`：REST 与 GraphQL 对比。
- `http_render_timeline`：HTTP 请求到页面渲染。
- `product_growth_metrics`：SaaS 增长指标体系。
- `tcp_failure_diagnosis`：TCP 连接失败排查路径。
